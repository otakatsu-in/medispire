# MediSpire Website Builder Prompt
## Use this prompt with Replit AI, Bolt.new, Lovable, or any AI website builder

---

## PROMPT (Copy Everything Below This Line)

---

Build a full-stack, SEO-optimized, content-heavy website called **MediSpire** (medispire.de) — a one-stop guidance platform for Indian healthcare professionals planning to move to Germany. The website is founded by **Dr. Sandeep Amin** (Founder & CEO) and **Dr. Sangeeta Pai** (Co-Founder & COO), both practicing senior medical specialists in Germany with 10+ years of experience helping Indian professionals navigate the German healthcare system.

---

## TECH STACK & SETUP

- **Frontend:** React.js with Next.js (for SSR/SEO) or plain HTML/CSS/JS if simpler
- **Styling:** Tailwind CSS — clean, minimal, professional medical aesthetic
- **Color Palette:** White (#FFFFFF) primary, deep navy blue (#1A2E4A) for headers/CTAs, gold/amber (#D4A017) for accents, light grey (#F5F7FA) for section backgrounds
- **Typography:** Inter or Poppins — clean, highly readable
- **Icons:** Lucide React or FontAwesome
- **Animations:** Subtle scroll-fade-in, hover effects — nothing flashy
- **Mobile:** Fully responsive, mobile-first
- **SEO:** Every page must have meta title, meta description, Open Graph tags, canonical URL, schema markup (Organization, FAQPage, BreadcrumbList)
- **Performance:** Lazy loading for images, optimized fonts

---

## NAVIGATION / HEADER

Sticky top navigation bar with:
- Logo: "MediSpire" with tagline "Your Key to Healthcare in Germany"
- Nav links: Home | About Us | Why Germany | For Doctors | For Dentists | For Nurses & Allied | Services | Blog | Resources | Contact Us
- Prominent CTA button: **"Book Free Consultation"** → opens booking modal
- Mobile hamburger menu
